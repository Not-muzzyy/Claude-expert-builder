# Knowledge: Learning Paths

## Cybersecurity (SOC / Analyst)

**Beginner**
- Linux basics (files, permissions, networking commands)
- Networking fundamentals (TCP/IP, DNS, HTTP, firewalls)
- Platform: TryHackMe Pre-Security path (free)

**Intermediate**
- SIEM tools (Splunk, ELK basics)
- Log analysis and alert triage
- CompTIA Security+
- Platform: TryHackMe SOC Level 1 path

**Advanced**
- Threat hunting, incident response
- CEH or OSCP for offensive skills
- Build projects: Mini-SIEM, phishing detector, log analyzer

**Practice Platforms:** TryHackMe, HackTheBox, CyberDefenders, BlueTeamLabs

---

## Machine Learning / AI

**Beginner**
- Python proficiency first (Pandas, NumPy)
- ML fundamentals: supervised vs unsupervised, train/test split, overfitting
- Resource: Kaggle Learn (free, fast)

**Intermediate**
- Scikit-learn full pipeline (preprocessing, models, evaluation)
- Deep learning basics (PyTorch or TensorFlow)
- LLM apps with Groq / HuggingFace

**Advanced**
- RAG systems, fine-tuning, embeddings
- MLOps (model versioning, deployment pipelines)
- Research papers on arXiv

**Practice Platforms:** Kaggle, HuggingFace, Google Colab

---

## Web Development

**Beginner**
- HTML + CSS (freeCodeCamp Responsive Web Design — free)
- Basic JavaScript (variables, functions, DOM)

**Intermediate**
- React basics
- Flask or FastAPI for backend
- Connect a database (SQLite → PostgreSQL)

**Advanced**
- Auth systems, REST + GraphQL APIs
- Docker, CI/CD with GitHub Actions
- Deploy to Render/Railway

**Practice:** Build real projects, contribute to open source

---

## Python (General)

**Beginner** → Variables, loops, functions, files, error handling
**Intermediate** → OOP, modules, virtual environments, APIs, Pandas
**Advanced** → Async, decorators, metaclasses, testing, packaging

**Resources:** Python.org docs, Real Python, Automate the Boring Stuff (free online)

---

## After BCA (Higher Studies)

- **MCA** — Master of Computer Applications (India, university route)
- **MSc Computer Science** — more research-oriented
- **Online Masters** — BITS Pilani WILP (work + study), OMSCS at Georgia Tech
- **Certifications as alternative** — Security+, OSCP, Google Cloud can substitute for formal degrees in many tech roles
